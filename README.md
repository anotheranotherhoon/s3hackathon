- [ ] Storybook 빌드하기 <br/> ```npm run build-storybook```
- [ ] 여러분의 Github에 새로운 레포지토리를 생성하고 storybook-static 폴더 안에 있는 파일들을 업로드합니다.
- [ ] Storybook 배포하기 **Netlify를 사용해보는 것을 추천드립니다.**<sup> 가능하면 Github 아이디로 가입해주세요.</sup>
